## build and run
Use the commands below to build and run the software contained in this repository

| command    |  Notes    |
| :------------------------ |  :-------  |
| gradlew build | Performs a compile and packages a jar file|
| gradlew run -Prc Roboomath | Performs a build step then runs the class Robomath.  Any class can be use with the -Prc property to run any class in this project.|
